title: "Mettre à jour le profil partenaire et les données société"
description: "Modifier raison sociale, adresse et téléphone depuis Paramètres. L’e-mail est en lecture seule — modification via le support HeyZack."
---

La page **Paramètres** permet de tenir à jour les informations du compte partenaire : raison sociale, adresse et numéro de téléphone. Les changements sont visibles immédiatement après enregistrement.

## Accès

**Paramètres** dans la barre latérale. La section **Général** s’affiche par défaut.

## Champs modifiables

Carte **Profil société** :

| Champ | Description |
|---|---|
| **Raison sociale** | Nom de l’entreprise partenaire enregistré chez HeyZack |
| **Adresse** | Adresse postale ou siège (zone de texte multiligne) |
| **Téléphone** | Numéro de contact professionnel |

<Note>
  L’**e-mail** est affiché mais ne peut pas être modifié dans le portail. Pour changer l’adresse enregistrée, contactez le support HeyZack.
</Note>

## Enregistrer

<Steps>
  <Step title="Ouvrir Paramètres">
    **Paramètres** dans la barre latérale.
  </Step>
  <Step title="Modifier">
    Ajustez **Raison sociale**, **Adresse** et/ou **Téléphone** — une ou plusieurs modifications dans la même sauvegarde.
  </Step>
  <Step title="Enregistrer les modifications">
    Bouton **Enregistrer les modifications** en bas à droite de la carte Profil société.
  </Step>
</Steps>

Sans modification sur les champs éditables, le portail peut afficher « Aucune modification à enregistrer ». Seuls nom, adresse et téléphone déclenchent une sauvegarde — pas l’e-mail.

<Tip>
  Une notification confirme le succès. En cas d’erreur, vérifiez la connexion puis réessayez ou contactez le support.
</Tip>
