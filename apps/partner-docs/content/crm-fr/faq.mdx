---
title: "HeyZack CRM : questions fréquentes"
description: "Comptes, rôles, chat, leads, jetons de parrainage et fonctionnalités du portail HeyZack CRM."
---

Réponses aux questions les plus courantes des utilisateurs du portail. Pour toute autre demande, contactez votre chargé de compte HeyZack ou votre administrateur.

<AccordionGroup>
  <Accordion title="Comment réinitialiser mon mot de passe ?">   
    Votre administrateur peut aussi envoyer un lien depuis **Paramètres → Réinitialiser le mot de passe**. Les liens expirent après 60 minutes — en cas d’expiration, demandez-en un nouveau.
  </Accordion>

  <Accordion title="Qu’est-ce qu’un code partenaire ?">
    Identifiant unique attribué à chaque partenaire. Il sert lors de la génération de jetons de lead pour rattacher un parrainage à un partenaire et pour le reporting et les commissions. Le code figure sur la fiche partenaire.
  </Accordion>

  <Accordion title="Puis-je avoir plusieurs rôles ?">
    Non. Chaque compte a un seul rôle parmi **Administrateur**, **Manager** ou **Support**. Ce rôle détermine les sections et actions accessibles. Pour plus de droits, demandez à un administrateur de modifier votre rôle dans **Paramètres → Gérer les rôles**.
  </Accordion>

  <Accordion title="Que se passe-t-il quand je génère un jeton pour un lead ?">
    Le portail crée un jeton de parrainage unique liant le lead au partenaire. Vous pouvez le transmettre au partenaire pour le suivi. Le jeton sert au reporting et aux commissions.
  </Accordion>

  <Accordion title="Comment fonctionne le chat en temps réel ?">
    Les messages sont diffusés dès l’envoi, sans rechargement de page. L’historique est conservé côté serveur. Si le temps réel cesse de fonctionner, vérifiez la connexion et consultez le [guide de dépannage](/troubleshooting).
  </Accordion>

  <Accordion title="Un partenaire et un technicien peuvent-ils chatter sans être connectés ?">
    Non. Le chat partenaire ↔ technicien exige une installation ou un chantier commun. Sans lien, l’option de chat n’apparaît pas.

    Les conversations partenaire ↔ admin et technicien ↔ admin n’ont pas cette contrainte — les administrateurs peuvent ouvrir une discussion avec n’importe quel partenaire ou technicien.
  </Accordion>

  <Accordion title="Comment changer la langue du portail ?">
    Utilisez le sélecteur de langue dans la barre latérale. Langues disponibles : **English (EN)** et **French (FR)**. Le choix est mémorisé pour les prochaines connexions.
  </Accordion>

  <Accordion title="Quels formats d’image sont pris en charge dans le chat ?">
    **JPEG**, **PNG**, **WebP** et **GIF**. La taille maximale peut être configurée par instance — renseignez-vous auprès de votre administrateur.
  </Accordion>

  <Accordion title="Puis-je exporter les données de vente ?">
    Oui. La page Rapport des ventes propose **Exporter en CSV**. Définissez la plage de dates et le filtre partenaire, puis exportez. Si le bouton est absent, votre compte n’a peut‑être pas la permission Rapport des ventes — voir avec l’admin.
  </Accordion>

  <Accordion title="Qui peut voir les journaux d’activité ?">
    Seuls les utilisateurs disposant de la permission **Journaux d’activité** y accèdent. Les administrateurs y ont toujours accès. Demandez à votre administrateur d’ajuster les permissions dans **Paramètres** si besoin.
  </Accordion>
</AccordionGroup>
