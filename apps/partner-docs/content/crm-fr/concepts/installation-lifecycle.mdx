---
title: "Comprendre le cycle de vie d’une installation"
description: "Étapes d’une installation maison connectée dans le CRM, du brouillon à l’accord client ou à l’annulation."
---

Une installation représente un chantier maison connectée — de sa création jusqu’à la validation par le client. Le CRM suit un ensemble de statuts pour que administrateurs, managers et partenaires sachent toujours où en est le dossier.

## Étapes du cycle

<Steps>
  <Step title="Brouillon">
    Installation créée mais pas encore planifiée. Saisie des informations de base — titre, client, partenaire — avant confirmation de date.
  </Step>
  <Step title="Planifiée">
    Date d’intervention fixée. En attente d’affectation technicien.
  </Step>
  <Step title="Affectée">
    Technicien affecté. Le chantier peut démarrer.
  </Step>
  <Step title="En cours">
    Technicien sur place, travaux en cours.
  </Step>
  <Step title="Terminée">
    Travaux terminés côté terrain. En attente de validation client.
  </Step>
  <Step title="Accord client">
    Le client a validé l’installation. Chantier clôturé.
  </Step>
  <Step title="Annulée">
    Installation annulée avant fin. Aucune action ultérieure.
  </Step>
</Steps>

<Note>
  Filtrez les installations par statut dans la vue **Installations** avec la liste **Filtrer par statut** à côté de la recherche.
</Note>

## Changements de statut

Les transitions se font depuis la fiche détail d’une installation. Ouvrez une ligne de la liste, puis utilisez les contrôles de statut sur la fiche pour faire avancer le dossier.

## Champs principaux

| Champ | Description |
|-------|-------------|
| Titre | Nom ou courte description du chantier |
| Client | Client final bénéficiaire de l’installation |
| Partenaire | Partenaire ayant apporté ou vendu le dossier |
| Technicien | Intervenant affecté |
| Date prévue | Date planifiée de passage |
| Statut | Étape courante du cycle |
